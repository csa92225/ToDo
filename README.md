# ToDo

To do list for my own practice using React.js

### Run:

```json
cd server
npm install
npm run dev
```

```json
cd client
npm install
npm run start
```

OR

### Alternative way to run with docker-compose:

```json
docker-compose -f ./docker-compose.yml up
```
